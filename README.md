# Smart Parking Finder

A responsive front-end web application designed to help campus users quickly find, compare, and reserve parking. The system is built around a simple goal: reduce the time, confusion, and frustration involved in searching for parking on a busy campus.

## Project Overview
Smart Parking Finder is a campus parking interface for three main user groups:
- **Students** looking for fast, affordable parking near classes
- **Staff/Faculty** needing reliable permit and lot access
- **Visitors** who need clear directions, parking rules, and temporary parking options

The application brings key parking tasks into one interface, including lot discovery, availability checking, reservations, permit information, alerts, and help resources.

## Main Features
- **Landing page with live campus parking summary**
- **Quick search** by building or destination
- **Interactive parking map section**
- **Lot filtering and availability display**
- **Reservation form and reservation summary**
- **Dashboard** for active, upcoming, and past reservations
- **Permit section** with permit status, permit types, and application flow
- **Notifications panel** for alerts and updates
- **Help/FAQ section** for common parking questions
- **Responsive layout** for desktop and mobile use

## System Purpose
Parking on campus can be stressful because users often do not know:
- which lots are available,
- what rules apply to each lot,
- where permits are accepted,
- how much parking costs,
- or whether a space can be reserved ahead of time.

Smart Parking Finder solves this by presenting parking information in a more organized and user-friendly way.

## Intended Users
### 1. Students
Need quick access to nearby parking before class and want to compare lot options based on distance, cost, and availability.

### 2. Staff and Faculty
Need dependable parking access, permit information, and a clear view of lot eligibility.

### 3. Visitors
Need simple guidance, temporary parking information, directions, and fewer confusing steps.

## Key User Tasks
- Search for parking near a destination
- View current lot availability
- Compare lot details and restrictions
- Reserve a parking space
- Review reservation details
- Check permit options and status
- View alerts and notifications
- Get answers through the FAQ/help section

## Design Goals
This project was designed to prioritize:
- **Clarity** — reduce confusion around parking rules and availability
- **Speed** — help users find a spot in seconds
- **Accessibility** — support a wide range of users and devices
- **Usability** — make important tasks easy to complete with minimal effort
- **Responsiveness** — provide a polished experience across screen sizes

## Current Scope
This repository contains the front-end prototype/interface for the Smart Parking Finder system. It demonstrates the user experience and interaction flow for the main parking tasks.

Depending on the course requirements, future versions could be extended with:
